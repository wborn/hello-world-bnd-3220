# hello-world-bnd-3220

This simple Maven project with a customized `<resources>` section in the POM builds properly in Eclipse when "Project -> Build Automatically" is enabled.

Whereas there is an infinite build loop when this customized `<resources>` section is added to a project using bnd such in this [autobuild-loop](https://github.com/wborn/jpmcli/tree/autobuild-loop) branch.

See also: [bndtools/bnd#3220](https://github.com/bndtools/bnd/issues/3220)
